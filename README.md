# Suunitelma
Tehdä esimerkkien mukaan kaksi kaverilista sovellusta. Ensin opiskella javascriptin käyttö youtubesta. Oppimisen jälkeen rakentaa sivut joille javascript tehdään jonka jälkeen js toimivaan kuntoon ja viimeisenä hieman hienostelua cssällä.

# Raportti
## Tavoitteet
Oppia Js perusteet ja tehdä tehtävän annon mukaiset sivut.
## Saavutus
12 tunnin pituisen js perusteita opettavan videon katsominen ja netistä tiedon etsiminen.
## Ilmentyneet ongelmat
Listojen erillaisuus verrattuna aiemmin käytettyyn pythoniin aiheutti nimien poistossa vaikeuksia. Nimien uudelleen tulostus kun listaan lisättiin uusi nimi.
## Opittu?
Kuinka javascript toimii varsinkin verrattuna pythoniin.
